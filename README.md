# Dataset Instructions

## CarPrice_Assignment.csv

This project uses the **Car Price Prediction** dataset from Kaggle.

### Download Steps

1. Go to: https://www.kaggle.com/datasets/hellbuoy/car-price-prediction
2. Click **Download** (requires free Kaggle account)
3. Extract and place `CarPrice_Assignment.csv` in the **root of this repository**

### Dataset Details

| Property | Value |
|---|---|
| Records | 205 cars |
| Features | 26 columns |
| Target | `price` (USD) |
| Source | Kaggle / UCI ML Repository |

### Key Columns

| Column | Description |
|---|---|
| `CarName` | Brand and model name |
| `fueltype` | gas / diesel |
| `aspiration` | std / turbo |
| `carbody` | sedan, hatchback, wagon, convertible, hardtop |
| `drivewheel` | fwd / rwd / 4wd |
| `enginesize` | Engine displacement (cc) |
| `horsepower` | Engine power (hp) |
| `curbweight` | Vehicle weight (lbs) |
| `citympg` | City fuel efficiency |
| `highwaympg` | Highway fuel efficiency |
| `price` | **Target variable** — car price in USD |

> **Note:** The dataset is not included in this repository due to Kaggle's licensing terms.  
> Please download directly from Kaggle using the link above.
